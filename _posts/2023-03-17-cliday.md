## Tip of the Week

Performing arithmetic in Bash:

```
% count=0
% ((count+=5))  # increment variable
% echo $count   # prints "5"

% i=$((count+5))
% echo $count   # prints "5"
% echo $i       # prints "10"

% echo "10 / 3" | bc            # prints "3"
% echo "10 / 3" | bc -l         # prints "3.33333333333333333333"
% echo "scale=2; 10/3" | bc -l  # prints "3.33"
% echo "3^2" | bc               # prints "9"
```
